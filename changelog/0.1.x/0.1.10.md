# 0.1.10 — 2026-06-02

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.9.16` → `^0.9.21` — adopts three framework fixes:
  - Per-request logs and traces now carry fresh request + trace/span IDs instead of the frozen boot context (HTTP transport log context fix).
  - **`fetchWithTimeout`** strips query-string secrets (e.g. `?api_key=`) from error messages and logs.
  - **`withRetry`** fails fast on non-retryable errors; `ctx.fail` auto-populates the `retryable` flag.
- **Skills synced** — 8 updated (`add-tool`, `api-canvas`, `api-context`, `api-linter`, `api-utils`, `design-mcp-server`, `release-and-publish`, `add-service`); new `api-mirror` and `orchestrations` skills added.
- **`scripts/release-github.ts`** added — `bun run release:github` creates the GitHub Release from the latest annotated tag.
- **`scripts/devcheck.ts`** updated — consolidated devcheck script.
- **README client-config keys** renamed to full package name `eurostat-mcp-server` (was `eurostat`).
